Building Blocks

As I track on API deployment, I'm trying to distill the services, and tooling down into what I consider to be common building blocks. My goal with identifying ApI deployment building blocks is to provide a simple list of the moving parts that make up the space.

Some of these building blocks overlap with other core areas of my research like design, and management, but I hope this list captures the basic building blocks of what anyone needs to know, to follow the world of API deployment.

If there are any features, service or tools you depend on when deploying your APIs, please let me know at @kinlane. I'm not trying to create an exhaustive list, just get an idea for what is available across the providers, and where the gaps are potentially.
